# Coverize

Generate a poster for your favorite album!

Enter a (valid) Spotify album URL, then download a poster!

## Running locally

1. Clone the repository
2. Install dependencies by running:

  ```shell
    bun install
  ```
   
3. Run the app by running:

  ```shell
    bun start
  ```

And voila!
